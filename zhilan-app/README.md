# Zhilan Idea King Frontend

This folder contains the Vite + React + TypeScript frontend for Zhilan Idea King.

For the full bilingual project overview, setup guide, current progress, roadmap, and open-source safety checklist, see:

```text
../README.md
```

## Quick Start

```powershell
npm install
npm run dev
```

Build:

```powershell
npm run build
```

Required local environment variables:

```env
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
VITE_LOCAL_SCRAPER_URL=http://127.0.0.1:8787
VITE_SEARCH_MODE=local
```

Do not commit `.env.local`.
